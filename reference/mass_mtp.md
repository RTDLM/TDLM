# Population and number of out- and in-commuters in Hérault municipalities, France (2020)

A dataset containing the number of inhabitants, in-commuters, and
out-commuters for 342 Hérault municipalities in 2020.

## Usage

``` r
mass_mtp
```

## Format

A `data.frame` with 342 rows and 4 columns:

- rownames:

  Municipality ID.

- Population:

  Number of inhabitants.

- Out_Commuters:

  Number of out-commuters.

- In_Commuters:

  Number of in-commuters.
