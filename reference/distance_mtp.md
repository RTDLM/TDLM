# Great-circle distances between Hérault municipalities, France

A dataset containing the great-circle distance (in kilometers) between
342 Hérault municipalities.

## Usage

``` r
distance_mtp
```

## Format

A `matrix` with 342 rows and 342 columns. Each element represents the
distance between two municipalities. Municipality IDs are used as row
and column names.

## Source

<https://www.data.gouv.fr/datasets/contours-communes-france-administrative-format-admin-express-avec-arrondissements/>
