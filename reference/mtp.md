# Spatial distribution of Hérault municipalities, France (2020)

A dataset containing the geometry of 342 Hérault municipalities.

## Usage

``` r
mtp
```

## Format

- ID:

  Municipality ID.

- Longitude:

  Longitude coordinate of the municipality centroid.

- Latitude:

  Latitude coordinate of the municipality centroid.

- Area:

  Surface area of the municipality (in square kilometers).

- geometry:

  Geometry of the municipality.

## Source

<https://www.data.gouv.fr/datasets/contours-communes-france-administrative-format-admin-express-avec-arrondissements/>
