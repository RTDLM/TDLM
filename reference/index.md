# Package index

## Utils

- [`check_format_names()`](https://rtdlm.github.io/TDLM/reference/check_format_names.md)
  : Check format of TDLM's inputs
- [`extract_distances()`](https://rtdlm.github.io/TDLM/reference/extract_distances.md)
  : Compute the distance between pairs of locations
- [`extract_opportunities()`](https://rtdlm.github.io/TDLM/reference/extract_opportunities.md)
  : Compute the number of opportunities between pairs of locations
- [`extract_spatial_information()`](https://rtdlm.github.io/TDLM/reference/extract_spatial_information.md)
  : Extract distances and surface areas from a spatial object
- [`calib_param()`](https://rtdlm.github.io/TDLM/reference/calib_param.md)
  : Automatic calibration of trip distribution laws' parameter

## Main functions

- [`run_law_model()`](https://rtdlm.github.io/TDLM/reference/run_law_model.md)
  : Estimate mobility flows based on different trip distribution laws
  and models
- [`run_law()`](https://rtdlm.github.io/TDLM/reference/run_law.md) :
  Estimate mobility flows based on different trip distribution laws
- [`run_model()`](https://rtdlm.github.io/TDLM/reference/run_model.md) :
  Estimate mobility flows based on different trip distribution models
- [`gof()`](https://rtdlm.github.io/TDLM/reference/gof.md) : Compute
  goodness-of-fit measures between observed and simulated OD matrices

## Data

- [`mass`](https://rtdlm.github.io/TDLM/reference/mass.md) : Population
  and number of out- and in-commuters in Kansas counties, USA (2000)
- [`distance`](https://rtdlm.github.io/TDLM/reference/distance.md) :
  Great-circle distances between Kansas counties, USA
- [`od`](https://rtdlm.github.io/TDLM/reference/od.md) :
  Origin-Destination commuting matrix for Kansas counties, USA (2000)
- [`county`](https://rtdlm.github.io/TDLM/reference/county.md) : Spatial
  distribution of Kansas counties, USA (2000)
- [`coords`](https://rtdlm.github.io/TDLM/reference/coords.md) :
  Geographical coordinates of Kansas counties' centroids
  (Longitude/Latitude, 2000)
- [`coords_xy`](https://rtdlm.github.io/TDLM/reference/coords_xy.md) :
  Geographical coordinates of Kansas counties' centroids (X/Y, Web
  Mercator, 2000)
