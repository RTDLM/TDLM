# Spatial distribution of Kansas counties, USA (2000)

A dataset containing the geometry of 105 Kansas counties.

## Usage

``` r
county
```

## Format

- ID:

  County ID.

- Longitude:

  Longitude coordinate of the county centroid.

- Latitude:

  Latitude coordinate of the county centroid.

- Area:

  Surface area of the county (in square kilometers).

- geometry:

  Geometry of the county.
