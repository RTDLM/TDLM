# Great-circle distances between Kansas counties, USA

A dataset containing the great-circle distance (in kilometers) between
105 Kansas counties.

## Usage

``` r
distance
```

## Format

A `matrix` with 105 rows and 105 columns. Each element represents the
distance between two counties. County IDs are used as row and column
names.

## Source

<https://www.census.gov/geographies/mapping-files/time-series/geo/carto-boundary-file.html>
